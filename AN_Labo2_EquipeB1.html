Details:

This project implements the equations solving dichotomy method. The resolution is as follows:
  • Searching for intervals containig roots
  • Finding the root in each interval
  • Calculate the relative error

If you want to solve your own function, here's the details:

The first input is for the function, but be aware that you have to write it in JavaScript (cf. RTFM). The independant variable is "x".
The second input has to be specified if the function have two roots whose interval is closer than 0.5 (e.g. high frequency sinus),
in order to search roots in smaller intervals (otherwise the function won't be able to find every root!).
